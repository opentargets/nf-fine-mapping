# Data generation

Notebooks to generate data for testing and benchmarking the fine-mapping pipeline.

1. `/bin/LD-interface.ipynb`: Generate LD reference files for testing the LD subsetting module. This creates the input channel for LD reference files.
2. `/bin/sushie-input-to-sumstat-output.ipynb`: Generate input files for testing the SuShiE module.
3. `/bin/manifest-generator.ipynb`: Notebook to generate manifest files for testing the pipeline. This is used to create the input channel for the pipeline, and can be used to generate different manifest files for testing different scenarios.
